Hello World，从零开始的博客之旅

发表于 2025-12-30 分类于 生活 阅读次数：

大家好，我是清茗心。

经过几天的折腾，我的个人博客终于上线了！
域名还是熟悉的 https://qingmingxin3245.github.io ，这次从头搭建，用了 Hexo + Next 主题，部署在 GitHub Pages 上，完全免费、干净简洁，也更符合我现在的风格——极简、专注内容、暗黑模式友好。

为什么重新建一个博客？

这次搭建新博客，我想留下一片纯粹的净土：

  • 写技术心得
  • 分享读书笔记和思考
  • 记录生活里那些转瞬即逝的灵感和情绪

没有广告，没有复杂插件，只留下文字和代码该有的样子。

关于这个博客

  • 框架：Hexo（静态博客生成器，轻快高效）
  • 主题：Next（经过自定义，暗黑优先、代码高亮友好）
  • 部署：GitHub Pages
  • 目标：坚持更新，成为自己的知识库和心灵角落

最后

这是一个全新的开始，也是一个对自己的承诺：
多写、多思考、多输出。

如果你路过这里，欢迎留言、欢迎交流。
希望这个小角落，能在未来的某一天，翻开旧文章时，让当时的自己会心一笑。

谢谢你看到这里

请我一杯咖啡吧！
